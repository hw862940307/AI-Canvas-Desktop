# 本地运行指南

为了方便您在本地运行此项目，我为您创建了两个简单的启动启动器：

### Windows 用户 (推荐)
1. **双击** 根目录下的 `启动项目.bat`。
2. 启动器会自动检查环境，并在需要时安装依赖。
3. 启动完成后，浏览器会自动打开或手动访问：`http://localhost:3000`。

### macOS / Linux 用户
1. 打开终端。
2. 首先赋予脚本执行权限：`chmod +x start.sh`。
3. 以后只需 **双击** `start.sh` 或在终端运行 `./start.sh` 即可。

### 注意事项：
*   **Node.js**: 请确保您的电脑已安装 Node.js (推荐 v18 或更高版本)。
*   **首次运行**: 第一次运行会因为安装依赖项（npm install）而稍微慢一些，之后启动会非常快。
